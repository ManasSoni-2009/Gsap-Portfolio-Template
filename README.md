Welcome to my high-performance, interactive 3D portfolio. This site showcases my work as a **Full-Stack Developer** and **AI Agent Specialist**, built using cutting-edge web technologies to provide a buttery-smooth, immersive user experience.

---

## ✨ Key Features

- **🎮 Reactive 3D Avatar**: A custom-rigged 3D character on the landing page that tracks your mouse movements and reacts to scroll events.
- **⚛️ Physics-Based Tech Stack**: An interactive "toy-box" of 3D spheres representing my skillset, powered by `@react-three/rapier` physics. 
- **📈 GSAP Narrative Scrolling**: Every section of the site is part of a timed animation sequence that transitions seamlessly as you scroll.
- **⚡ Performance Optimized**: Custom-padded SVG textures, simplified 3D geometries, and manual asset chunking ensure a fluid 60FPS experience even on mobile.
- **📱 Fully Responsive**: Tailored viewing experience across mobile, tablet, and desktop.

---

## 🛠️ Tech Stack

### Frontend & 3D
- **React 18** + **TypeScript**
- **Vite** (Next-gen build tool)
- **Three.js** (WebGL Engine)
- **React Three Fiber** (React bridge for Three.js)
- **Rapier** (High-performance 3D Physics)

### Animation & Styles
- **GSAP (GreenSock)** + **ScrollTrigger**
- **Vanilla CSS** (Custom theme system)
- **SplitText** (Cinematic typography)

## 🚀 Quick Start

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed (v18 or higher recommended).

### 2. Installation
```bash
# Clone the repository
git clone https://github.com/ManasSoni-2009/PORT-FM.git

# Install dependencies
npm install
```

### 3. Development
```bash
# Start the local dev server
npm run dev
```

### 4. Build for Production
```bash
# Generate optimized assets for GitHub Pages
npm run build
```
